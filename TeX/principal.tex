\begin{abstract}
\noindent
Este exercício prático contém a documentação do Trabalho Prático I, feita em LATEX. O problema de encontrar cada componente biconexo de um grafo, o que representa encontrar propriedades interessantes, sendo que, em algumas áreas da Computação, se diz que tal grafo é “tolerante a falhas”. Essa tarefa lida com a questão de encontrar todos os componentes biconexos, blocos, de um grafo conexo com tamanhos variados de vértices e arestas. Neste trabalho foram implementadas e comparadas três formas distintas para resolução do problema de encontrar blocos.  Uma forma utilizando um método que verifique a existência de dois caminhos internamente disjuntos (ou um ciclo) entre cada par de vértices do bloco. Outro método que identifica articulações testando a conectividade após a remoção de cada vértice. A última e melhor solução que garante a resposta com uma boa complexidade, utilizando o método de Tarjan \cite{artigo_tarjan}. Os códigos do trabalho prático foram implementados em um mesmo projeto utilizando a linguagem C++.  %por citação do tarjan
\\\textbf{\keyword{Palavras-chave: }} Componentes Biconexos. C++. Tarjan. Grafos.
\end{abstract}
